About

Gesture Runner is a browser-based game controlled using real-time hand tracking. It uses MediaPipe Hands to detect landmarks, then maps stable finger counts to game actions.

What it demonstrates

  • Real-time computer vision (hand landmarks)
  • Gesture recognition with stability filtering
  • Interactive gameplay + UI/UX design
  • Front-end engineering in the browser

Tech used

  • HTML + CSS + JavaScript
  • MediaPipe Hands
  • Canvas rendering
